# Инструкция по деплою на Netlify

## Настройки в Netlify Dashboard

1. **Build settings:**
   - **Base directory:** оставьте пустым (если проект в корне репозитория)
   - **Build command:** `npm run build` (или оставьте пустым, т.к. указано в netlify.toml)
   - **Publish directory:** `dist` (или оставьте пустым, т.к. указано в netlify.toml)

2. **Environment variables:**
   - Убедитесь, что **НЕ** установлена переменная `VITE_BASE_PATH` (для Netlify должен быть `/`)
   - Если она установлена, удалите её или установите значение `/`

3. **Deploy settings:**
   - После изменений нажмите **"Trigger deploy"** → **"Clear cache and deploy site"**

## Проверка после деплоя

1. Откройте сайт в браузере
2. Откройте консоль разработчика (F12)
3. Проверьте, нет ли ошибок загрузки ресурсов
4. Проверьте Network tab - все файлы должны загружаться с кодом 200

## Если все еще 404

1. Проверьте логи сборки в Netlify на наличие ошибок
2. Убедитесь, что файл `public/_redirects` существует
3. Проверьте, что в настройках Netlify не установлен неправильный базовый путь
4. Попробуйте удалить и пересоздать сайт в Netlify
